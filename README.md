# Codepresso

개인 포트폴리오 및 블로그를 개발하고 있습니다..  
개발 스택은 NodeJS, Express, ES6, RestAPI, Sequelize, MySQL, VueJS 입니다.  

Git Commit Prepix  
init - 프로젝트 최초 커밋  
dev - 새로운 기능 추가  
add - 커밋 누락  
rm - 파일 삭제  
fix - 간단한 수정  
bugfix - 버그 수정  
doc - 문서 작성  

프로젝트 진행에 관한 정보는 [Making Log](https://codepresso.net/devlog/topic/1) 에서 확인 할 수 있습니다.
